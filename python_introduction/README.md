Beginner Python Exercises
Overview

This repository contains a series of beginner-friendly Python exercises designed to help you practice fundamental programming concepts such as:

Variable assignments

Arithmetic operations

User input

Printing output

Each exercise is structured to gradually build your confidence in Python programming and provide hands-on experience with practical examples.

Exercises Included

1. Basic Arithmetic Operations

File: basic_operations.py

Description: Perform addition, subtraction, and multiplication on two predefined numbers and print the results.

2. Simple Interest Calculator

File: simple_interest.py

Description: Calculate simple interest using a given principal, rate, and time.

3. Rectangle Area Calculator

File: rectangle_area.py

Description: Compute the area of a rectangle using length and width.

4. Hours to Seconds Converter

File: hours_to_seconds.py

Description: Convert a given number of hours into seconds.

6. Finance Calculator

File: finance_calculator.py

Description: Calculate monthly savings from user input, and project annual savings with a 5% interest rate.

Learning Objectives

By completing these exercises, you will be able to:

Understand and assign variables in Python

Perform basic arithmetic operations

Accept and process user input

Display outputs in a clear, readable format

Apply Python concepts to simple real-world problems

Getting Started

Clone the repository or download the scripts.

Ensure Python is installed on your system.

Run any script using the command:

python3 script_name.py

Replace script_name.py with the specific file you want to run.

Example

Running basic_operations.py produces:

Addition of 10 and 5 is 15
Subtraction of 10 and 5 is 5
Multiplication of 10 and 5 is 50
